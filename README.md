# Programsko inženjerstvo - Tim 11.1 - Ekonomisti

## Opis projekta

Projekt predstavlja razvoj web platforme za računovodstvene servise koja omogućuje jednostavnu i sigurnu suradnju između računovođa, njihovih klijenata i radnika klijenata.

Platforma centralizira sve poslovne podatke na jednom mjestu — računovođe imaju pristup podacima svih svojih klijenata, dok svaki klijent vidi samo vlastite podatke i podatke svojih radnika.

Sustav koristi OAuth 2.0 autentifikaciju za prijavu korisnika, pri čemu se korisnici registriraju kao računovođa, klijent ili radnik. 

Računovođe mogu dodavati nove klijente, određivati mjesečne cijene svojih usluga i obrađivati dokumente koje klijenti i radnici unose putem platforme (putni nalozi, fakture). Nakon obrade podataka, računovođe izrađuju mjesečne financijske izvještaje (prihodi, rashodi, PDV, dobit/gubitak), koji se automatski šalju klijentima e-mailom zajedno s računom za usluge.
Klijenti putem platforme upravljaju svojim radnicima, financijskim podacima i plaćanjima, dok radnici imaju ograničene ovlasti i mogu unositi samo potrebne dokumente.
Cjelokupnim sustavom upravlja administrator, koji nadzire sve korisnike i može dodavati ili uređivati računovođe.

Cilj projekta je digitalizirati i pojednostaviti procese računovodstva, omogućiti transparentnu komunikaciju između klijenata i računovođa te automatizirati izradu i distribuciju financijskih izvještaja.
